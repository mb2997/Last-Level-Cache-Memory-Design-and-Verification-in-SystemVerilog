Steps to run:
1) Go to SIM directory and open the cmd terminal
2) make cmp -> only for compilation
3) make sim_arg TRACE_FILE=<INPUT_TRACE_FILE_NAME_WITH_EXTENSION> MODE=<NORMAL/SILENT> -> only for simulation (No compilation would be performed)
4) make all_arg TRACE_FILE=<INPUT_TRACE_FILE_NAME_WITH_EXTENSION> MODE=<NORMAL/SILENT> -> for compilation + simulation
